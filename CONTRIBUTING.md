
# Contributing

1. Keep `.proto` files the **single source of truth**.
2. Add language-specific generated code to **subfolders** (or ignore if regenerable).
3. For LabVIEW, commit only **VIs and projects** needed to run the examples.
4. Add **README.md** in each folder with quickstart steps.
5. Use PRs; include screenshots of LabVIEW block diagrams when helpful.
